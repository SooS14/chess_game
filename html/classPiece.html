Detailed Description

most important class of the program

This class contains all the methods that are related to the various pieces on the Chessboard. The virtual function "move" is declared here and has a different implementation for each piece. The attributes "p_first_move" and "p_en_passant" are common for all pieces despite the fact that some pieces don't need them (for instance the knight) but it makes the program clearer.
